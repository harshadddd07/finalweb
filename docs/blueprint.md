# **App Name**: MediSync Pro

## Core Features:

- Doctor Login: Secure login for doctors with role-based access control.
- Patient Login: Secure login for patients with role-based access control.
- New User Registration: Collect basic user information (Name, Age, etc.) during registration.
- Profile Management: Allow doctors and patients to manage their profiles.
- Appointment Scheduling: Enable patients to schedule appointments with doctors. Doctors see requests on their dashboard.
- Billing and Payment: Handle billing and payment processing, including UPI ID 'harshad@213' with QR code.
- Symptom Analysis Tool: Provide doctors with a tool that uses an LLM tool to help them with identifying diseases based on reported symptoms.
- Chat Section: Secure chat feature for real-time communication between patients and doctors.
- Video Call Section: Secure and integrated video call functionality for consultations.
- Live Chat on Doctor Dashboard: Live chat feature embedded directly within the doctor's dashboard for immediate patient communication.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), conveying trust and professionalism.
- Background color: Light cyan (#E0FFFF), provides a clean and calm interface.
- Accent color: Royal blue (#4169E1), used for calls to action and highlights.
- Body font: 'PT Sans', sans-serif, for clarity and readability.
- Headline font: 'Space Grotesk', sans-serif, for a modern, technical feel.
- Use clean, minimalist icons to represent different features and categories.
- Professional layout with a clear hierarchy and easy navigation.
- Subtle animations to enhance user experience and provide feedback.
